# EDA: Tendencias en Programación

Proyecto de **Análisis Exploratorio de Datos (EDA)** desarrollado como parte
del Bootcamp de Programación. El análisis estudia las tendencias actuales en
lenguajes de programación, perfiles de desarrolladores y salarios en el sector
tecnológico.

## Estructura del proyecto

```
EDA-programacion-tendencias/
├── notebooks/
│   └── eda_programacion_tendencias.ipynb   # Notebook principal con el EDA completo
├── requirements.txt                         # Dependencias Python
└── README.md
```

## Contenido del análisis

| Sección | Descripción |
|---------|-------------|
| 1. Carga de datos | Generación de datasets representativos a partir de informes públicos |
| 2. Exploración inicial | Estadísticas descriptivas y valores nulos |
| 3. Tendencias de lenguajes | Evolución del uso de 13 lenguajes (2017-2023) |
| 4. Perfiles de desarrolladores | Roles, experiencia y distribución geográfica |
| 5. Análisis de salarios | Salarios por rol, país y tamaño de empresa |
| 6. Modalidad de trabajo | Remoto, híbrido, presencial y SO preferido |
| 7. Aprendizaje continuo | Impacto del aprendizaje online en salarios |
| 8. Conclusiones | Síntesis de hallazgos clave |

## Principales hallazgos

- 🥇 **JavaScript** sigue siendo el lenguaje más usado, aunque **TypeScript** crece
  rápidamente y podría superarlo a medio plazo.
- 📈 **Python** mantiene su ascenso impulsado por IA/ML y Data Science.
- 🚀 **Go**, **Rust** y **Kotlin** son los lenguajes emergentes con mayor crecimiento.
- 💰 Los **ML Engineers** y **Data Scientists** son los roles mejor remunerados.
- 🌍 Los salarios en **USA** son notablemente superiores al resto de países.
- 🏠 El **65%** de los desarrolladores trabaja en remoto o en modalidad híbrida.
