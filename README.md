# EDA - Tendencias en Programación

Análisis exploratorio de datos de las encuestas de Stack Overflow Developer Survey (2021-2025).

## Datos

Los archivos CSV de las encuestas superan el límite de 100 MB de GitHub y no están incluidos en el repositorio.

### Descarga

1. Ve a la sección [Releases](https://github.com/jlalonsoredon/EDA-programacion-tendencias/releases) de este repositorio
2. Descarga los archivos CSV
3. Colócalos en la carpeta `data/`:

```
data/
├── stack-overflow-developer-survey-2021.csv
├── stack-overflow-developer-survey-2022.csv
├── stack-overflow-developer-survey-2023.csv
├── stack-overflow-developer-survey-2024.csv
├── stack-overflow-developer-survey-2025.csv
└── tendencias_lenguajes.csv
```

Alternativamente, puedes descargarlos directamente desde [Stack Overflow](https://survey.stackoverflow.co/).
